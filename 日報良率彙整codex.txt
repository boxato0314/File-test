2. **Excel 自動統計**
   自動整理每日、線別、站點、機種、批號等欄位，計算：
   - 投入數
   - 良品數
   - 不良數
   - 良率
   - 不良率
   - Top N 不良項目
   - 趨勢比較

3. **自動畫圖**
   產生每日良率趨勢圖、線別比較圖、不良 Pareto 圖等。

4. **貼到報告**
   可輸出成：
   - Excel 報表
   - Word 報告
   - PowerPoint 簡報
   - PDF
   - 或把圖表貼到既有報告模板中

- 不建議放 `TotalDies`，因為數量級太大

如果一定要看產出量，另外做一張量圖：

```text
良率圖：PanelYield / DieYield
產出圖：TotalDies / TotalPanels
```

**為什麼不要每個 OPNO 一張樞紐表**

假設你有 20 個 OPNO：

- 20 張樞紐表
- 20 個圖表
- 20 個 OPNO 篩選
- 每天新增日期時，可能有部分圖沒有吃到新日期
- 樞紐快取變多，檔案變慢

但如果用 `ChartData`：

- 只有一份資料來源
- 公式固定
- 圖表固定
- 每天只更新資料
- 新月份/新日期加進 `ChartData` 後，圖會跟著長

**最實用結論**

你的情境我會這樣做：

1. 巨集每天產生 `PivotData / tblYieldPivot`
2. 另外做 `ChartData_Month`
3. 每個 OPNO 兩欄：`PanelYield`、`DieYield`
4. Dashboard 每個 OPNO 一張小折線圖
5. 每天跑 `UpdateOPNOYieldDataOnly`
6. 圖表自動更新，不碰樞紐表篩選

這是最適合「很多 OPNO、每日更新、固定報告圖表」的方式。

> 上述是否就無法創建很多圖

可以創建很多圖，**但不要用同一張樞紐圖靠 OPNO 篩選器來複製很多張**，那樣每張圖其實會綁同一套樞紐篩選，維護會很痛苦。

你有兩條路：

**方案 A：少量圖，用多張樞紐圖**
如果只有 3 到 5 個 OPNO，可以：

- 每個 OPNO 一張樞紐表
- 每張樞紐表篩選固定一個 OPNO
- 每張樞紐表各自產生一張樞紐圖

可以做，但 OPNO 一多就不建議。

**方案 B：很多圖，用 ChartData，推薦**
如果你要 `A5AC004、A5FT001、A5FT002、A5SC002...` 很多張固定圖，建議不要靠樞紐圖，而是：

```text
PivotData / tblYieldPivot
        ↓
ChartData
        ↓
很多普通折線圖
```

每張普通圖只吃 `ChartData` 裡對應 OPNO 的資料區。
這樣每張圖彼此獨立，不受樞紐篩選器影響，每天更新也比較穩。

例如：

| Period | A5AC004 PanelYield | A5AC004 DieYield | A5FT001 PanelYield | A5FT001 DieYield |
|---|---:|---:|---:|---:|
| 2026-01 | 98.1% | 99.9% | 96.5% | 99.8% |
| 2026-02 | 97.8% | 99.9% | 95.2% | 99.7% |

然後：

- 圖 1：Period + A5AC004 兩欄
- 圖 2：Period + A5FT001 兩欄
- 圖 3：Period + A5FT002 兩欄

這樣你可以建立很多圖，而且每天只要更新 `ChartData`，圖會跟著更新。

簡單講：

- 想互動查詢：用一張樞紐圖 + OPNO 篩選器
- 想固定報告、很多 OPNO 圖：用 `ChartData` + 普通圖表

你的情境比較像固定報告，所以我建議走 **ChartData + 多張普通圖表**。

Excel 的主座標 / 副座標其實只影響 Y 軸刻度，不應該影響 X 軸位置。
如果柱狀圖跟折線圖 X 位置不一致，通常是下面幾個原因。

**最常見原因 1：系列範圍不同**

例如折線圖抓：

```text
Period: A2:A6
PanelYield: K2:K6
DieYield: L2:L6
```

但柱狀圖抓：

```text
TotalPanels: M2:M10
```

這樣 Excel 會用不同長度的資料去對應類別，柱就會錯位。

每個系列都要是同一段列，例如都抓 5 列：

```text
X 軸：A2:A6
PanelYield：K2:K6
DieYield：L2:L6
TotalPanels：M2:M6
```

**最常見原因 2：空白列或 #N/A 處理不同**

良率線如果遇到 `#N/A`，Excel 會跳過該點。
柱狀圖如果是空白或 0，Excel 仍可能畫出類別位置。

所以看起來會像：

- 線圖只畫有值的日期
- 柱狀圖仍佔所有日期位置
- 兩者視覺上不同步

這不是副座標問題，是不同系列對空值的處理方式不同。

**最常見原因 3：使用了「類別次序反轉」**

你目前資料是最新在上：

```text
2026-05-10
2026-05-09
2026-05-08
...
```

然後用類別次序反轉讓圖表顯示舊到新。

這在單純折線圖通常沒問題，但組合圖加副軸柱狀時，有時會讓柱圖視覺位置跟資料點感覺不一致，尤其資料範圍不是滿列或有空值時。

**最常見原因 4：柱狀圖在副座標軸後，Excel 產生第二組類別設定**

有些 Excel 版本在組合圖中，副軸系列會有自己的繪圖邏輯。雖然 X 軸看起來同一個，但當資料列數、空白處理、反轉方向不同時，副軸柱圖容易看起來「沒有跟線共用同一個 X 軸」。

**你可以先這樣檢查**

右鍵圖表 → `選取資料`

逐一點三個系列：

- `A5AC004 PanelYield`
- `A5AC004 DieYield`
- `A5AC004 TotalPanels`

確認每個系列的列範圍完全一致。

例如都應該是：

```text
A5AC004 PanelYield = ChartData_Day!$K$2:$K$6
A5AC004 DieYield   = ChartData_Day!$L$2:$L$6
A5AC004 TotalPanels = ChartData_Day!$M$2:$M$6
X軸標籤             = ChartData_Day!$A$2:$A$6
```

不要讓 TotalPanels 抓到 `M2:M10`，而良率只抓 `K2:K6`。

**我的判斷**

你的副座標柱狀圖錯位，最大可能是：

```text
TotalPanels 系列範圍筆數 ≠ PanelYield / DieYield 系列範圍筆數
```

或是圖表還保留了舊範圍，當資料天數變少時，折線系列變短，但柱狀系列還抓舊的多列。

所以重點不是副座標不能連動，而是要讓所有系列：

```text
同一個 Period 範圍
同一個起始列
同一個結束列
同一個空值處理方式
```

你可以先從 `選取資料` 裡檢查這三個系列範圍，通常一看就會找到哪一條範圍不一致。

> 我使用了「類別次序反轉」

那就很可能是它造成的。

`類別次序反轉` 在單純折線圖通常沒問題，但在你的圖這種：

- 折線圖：`PanelYield / DieYield`
- 柱狀圖：`TotalPanels`
- 柱狀圖放副座標軸
- 資料列有時不足固定圖表範圍
- 有些 OPNO 某些日期沒有資料

這幾個條件疊在一起時，Excel 很容易讓柱狀圖看起來沒有跟折線圖同步。

**為什麼會這樣**

你的資料表是最新在上：

```text
2026-05-10
2026-05-09
2026-05-08
2026-05-07
2026-05-06
```

你勾 `類別次序反轉` 後，X 軸顯示變成：

```text
2026-05-06
2026-05-07
2026-05-08
2026-05-09
2026-05-10
```

但組合圖裡的柱狀系列、折線系列、資料標籤、副座標軸，有時不會用完全一致的方式處理這個反轉，尤其當系列範圍長度不同或有空白值時，就會出現錯位感。

**比較穩的圖表設定**

如果你先不改 VBA，可以這樣試：

1. 不要勾 `類別次序反轉`
2. 在 `ChartData_Day` 旁邊手動做一塊圖表用資料區
3. 用公式把最新 N 天反向排列成舊到新
4. 圖表接這塊資料區

例如原始是：

```text
A2:A11 = 2026-05-10 到 2026-05-01
```

旁邊做：

```text
AA2:AA11 = 2026-05-01 到 2026-05-10
```

公式可以用：

```excel
=INDEX($A$2:$A$11, ROWS($A$2:$A$11)-ROW(A1)+1)
```

對數值欄也同樣反向抓。

這樣圖表本身不用反轉，副座標柱狀圖會穩很多。

**如果只靠設定**

你可以先檢查三件事：

1. `PanelYield`、`DieYield`、`TotalPanels` 三個系列範圍列數完全一致
2. `選取資料 > 隱藏和空白儲存格` 裡，空白儲存格不要用「連接資料點」
3. X 軸取消 `類別次序反轉` 測試一次

如果取消反轉後柱狀圖正常，幾乎就可以確認問題是 `類別次序反轉 + 組合圖副軸` 的 Excel 行為。

**我的建議**

長期最穩的是：

- `ChartData_Day` 保持最新在上，方便你看資料
- 另外做一塊「圖表專用反向資料區」
- 圖表接反向資料區
- 圖表不要使用 `類別次序反轉`

這樣可以同時滿足：

- 資料大於圖表範圍：抓最新 N 筆
- 圖表顯示：由舊到新
- 副座標柱狀圖：不錯位

> 有機會移動選取範圍時，圖表格式不要變嗎
>
> User context
>
> Images:
> - `~/AppData/Local/Packages/MicrosoftWindows.Client.Core_cw5n1h2txyewy/TempState/ScreenClip/{7639EAD9-C9F1-4FB2-9E53-7D54D2AEC1CC}.png`

可以，而且你現在遇到的是 Excel 的典型行為：**移動圖表資料範圍後，Excel 會重新套用系列設定**，尤其是你從 `A5TC001` 移到 `A5SC002`，它可能把：

- 顏色
- 主/副座標軸
- 圖表類型
- 資料標籤
- 座標軸刻度

重新判斷一次，所以格式跑掉。

最有效率的做法有三種。

**方法 1：先存成圖表範本，最簡單**

調好一張標準圖後：

1. 右鍵圖表
2. `另存為範本`
3. 存成例如：

```text
YieldCombo.crtx
```

之後每次圖表格式跑掉：

1. 右鍵圖表
2. `變更圖表類型`
3. `範本`
4. 選 `YieldCombo`

這會把圖表類型、顏色、副軸、線條等格式拉回來。

**方法 2：複製已完成圖表，再只改資料來源**

這比重新建圖穩：

1. 先完成一張 `A5TC001` 圖
2. 複製整張圖
3. 貼到旁邊
4. 右鍵圖表 → `選取資料`
5. 只改三個系列的值範圍：
   - `TotalPanels`
   - `PanelYield`
   - `DieYield`
6. 不要重新插入圖表

這樣格式通常能保留 80% 以上。

**方法 3：用輔助資料區，圖表永遠不改來源，最穩**

這是最推薦的長期方式。

建立一塊固定圖表資料區，例如：

| Period | TotalPanels | PanelYield | DieYield |
|---|---:|---:|---:|
| 2026-W10 | 20 | 87.71% | 99.71% |
| 2026-W11 | 30 | 93.55% | 99.94% |

圖表永遠吃這四欄。
你要換 OPNO 時，只改這塊資料區的公式，不改圖表本身。

例如在某個儲存格放 OPNO：

```text
B1 = A5SC002
```

資料區公式用 `XLOOKUP` 或 `INDEX/MATCH` 去抓：

```excel
=XLOOKUP($B$1 & " TotalPanels", ChartData_Week!$1:$1, ChartData_Week!$2:$20)
```

這樣圖表的來源範圍永遠不變，所以格式完全不會跑。

**我的建議**

你現在如果是在手動大量做圖：

- 短期：用 **方法 1 + 方法 2**
- 長期：做 **方法 3**

也就是：

```text
ChartData_Week
        ↓
固定輔助資料區
        ↓
固定圖表
```

你不要一直移動圖表選取範圍。
一移動，Excel 就可能重算圖表系列，格式就會變。

最穩的一句話是：

**圖表來源範圍固定，讓公式去換資料，不要讓圖表本身換範圍。**

> 有沒有excel設定值可以改

有幾個 Excel 設定可以調，但我要先講結論：**沒有一個全域設定可以保證「改資料範圍後圖表格式完全不變」**。不過下面這些設定可以降低格式跑掉的機率。

**1. 關閉自動套用新格式**

到：

```text
檔案 > 選項 > 進階
```

找到圖表相關設定，取消勾選類似這個選項：

```text
內容變更時自動擴充資料範圍格式與公式
```

英文可能是：

```text
Extend data range formats and formulas
```

這比較偏資料表/範圍格式，不一定完全解決圖表系列格式，但可以減少 Excel 自動判斷。

**2. 圖表不要使用「切換列/欄」後再改範圍**

如果曾按過：

```text
圖表設計 > 切換列/欄
```

Excel 很容易在你移動範圍時重判斷系列。
建議固定一種方向：

- X 軸：`Period`
- 系列：各欄位，例如 `TotalPanels / PanelYield / DieYield`

不要頻繁切換。

**3. 使用「選取資料」逐一改系列，不要拖曳彩色框**

你現在用拖曳彩色框移動範圍，Excel 最容易重套格式。

比較穩的是：

```text
右鍵圖表 > 選取資料 > 編輯系列
```

逐一改：

- 系列名稱
- 系列值
- 水平軸標籤

不要用滑鼠拖整個彩色框。

**4. 鎖定圖表類型與座標軸**

改完資料來源後，馬上確認：

```text
圖表設計 > 變更圖表類型 > 組合圖
```

設定：

| 系列 | 類型 | 座標軸 |
|---|---|---|
| TotalPanels | 群組直條圖 | 副座標軸 |
| PanelYield | 折線圖 | 主座標軸 |
| DieYield | 折線圖或只有標記 | 主座標軸 |

這是目前最接近「固定格式」的手動設定。

**5. 檢查資料標籤與空白資料設定**

右鍵圖表 → `選取資料` → `隱藏和空白儲存格`

建議：

```text
空白儲存格顯示為：空距
```

不要選「以零值顯示」，否則柱圖/線圖看起來更容易怪。

**最實際建議**

如果你問的是「有沒有像設定開關一樣，讓我移動範圍但圖表格式不變」：

```text
沒有可靠的 Excel 設定
```

- 不要拖曳彩色框

只改欄位 `$AL$2:$AL$11`，不要重建系列。這樣格式最不容易跑。
