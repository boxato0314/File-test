PowerPoint 自動合併工具－使用簡介

本工具使用一個空白的 PowerPoint 巨集簡報作為執行程式，自動搜尋該簡報所在資料夾及所有子資料夾內的 PowerPoint 檔案，並依照資料夾路徑與檔名順序合併成一份新的簡報。

使用方法：

1. 建立一個空白 PowerPoint。
2. 將檔案另存為「PowerPoint 啟用巨集簡報（.pptm）」。
3. 將此 .pptm 檔案放在需要合併資料的最上層資料夾。
4. 開啟 .pptm 檔案後，若 PowerPoint 顯示安全性警告，請先按下「啟用內容」或「啟用巨集」。
5. 按下鍵盤 Alt + F8，開啟巨集執行視窗。
6. 選擇「MergeAllPowerPointFiles」。
7. 按下「執行」。
8. 程式會自動搜尋目前資料夾及所有子資料夾內的 PowerPoint 檔案。
9. 程式找到檔案後會顯示檔案數量，按下「是」即可開始合併。
10. 合併完成後，程式會在執行工具所在的相同資料夾中，產生一份新的合併簡報。

支援格式：

本工具支援 PPT、PPTX、PPTM、PPS、PPSX、PPSM 等 PowerPoint 檔案格式。

程式會自動排除執行用的 .pptm 工具檔案、Office 暫存檔，以及先前產生的合併結果，避免重複合併。

輸出檔案：

合併完成後，輸出檔案名稱格式為：

Merged_Presentation_年月日_時分秒.pptx

例如：

Merged_Presentation_20260723_103500.pptx

合併順序：

程式會依照「完整資料夾路徑及檔名」由小到大排序後進行合併。

建議在資料夾名稱及 PowerPoint 檔案名稱前加入編號，以確保合併順序符合需求。

例如：

01_專案介紹.pptx

02_問題分析.pptx

03_改善對策.pptx

04_改善結果.pptx

注意事項：

執行工具必須先儲存為 .pptm 格式，否則無法保留及執行 VBA 巨集。

執行工具應放在需要搜尋的最上層資料夾，程式才會搜尋該資料夾及其下方所有子資料夾。

合併前請確認來源 PowerPoint 檔案可以正常開啟，且沒有加密、密碼保護、檔案損壞或權限限制。

執行期間請勿關閉 PowerPoint，並等待程式顯示「合併完成」。

合併後產生的檔案為一般 .pptx 格式，不包含 VBA 巨集。

若部分檔案無法合併，程式會自動跳過該檔案，並在完成訊息中顯示失敗檔案清單。
